What is Bitcoinbank?
----------------

Bitcoinbank is a lite version of Bitcoin using scrypt as a proof-of-work algorithm
 - 2 minute block targets
 - 210 million total coins
 - 50 coins per block
 - Coin subsity is halved every 840,000 blocks or approximately 3 years
